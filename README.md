# 📖 小故事家 · 逻辑表达训练营

为中国小学生（6岁+）设计的亲子逻辑表达训练静态网站。

## 文件结构

```
/
├── index.html        首页：六步框架图示 + 年级入口
├── grade1-2.html     1–2年级故事库（30个故事，简化3步）
├── grade3-4.html     3–4年级故事库（30个故事，完整6步）
├── grade5-6.html     5–6年级故事库（30个故事，6步+元认知）
├── planner.html      每日计划 + 计时器 + 打卡日历 + 成长追踪
└── README.md         本文件
```

## 🚀 如何使用

1. 获取链接：https://yongluy.github.io/little_storyteller/index.html
2. 把这个链接发到微信群，家长点击即可直接打开，无需任何安装。
3. 有任何问题或建议，请留言给我


## 内容概览

| 年级 | 框架深度 | 故事数 | 每次时长 | 目标 |
|------|---------|-------|---------|------|
| 1–2年级 | 简化3步 | 30个 | 5–8分钟 | 听懂·说出来 |
| 3–4年级 | 完整6步 | 30个 | 8–10分钟 | 说清楚·有逻辑 |
| 5–6年级 | 6步+元认知 | 30个 | 10–15分钟 | 有观点·能创作 |

**12大内容维度**：自然与生命、友谊与合作、勇气与成长、逻辑与解题、文化与多元、情绪与同理心、家庭与社区、科学与探索、创意与想象、公平与责任、数字与科技、历史与传统

## 特点

- ✅ 纯静态，无需服务器，无需登录
- ✅ 微信内点链接直接打开
- ✅ 移动端适配，阅读友好
- ✅ 进度数据保存在本地浏览器
- ✅ 维度筛选、故事弹窗、计时器、打卡日历、徽章系统
